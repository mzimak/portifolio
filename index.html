Modelagem de dados

Modelagem em bancos de dados é o processo de criar um esquema que define como os dados serão coletados, gerenciados e armazenados.

  • DbSchema
  • Toad Data Modeler
  • Lucidchart

Limpeza e Enriquecimento de Dados

O enriquecimento de dados consiste em complementar e aprimorar os dados que você já possui, agregando novas informações que ajudam a construir uma visão mais completa do cliente ou do mercado.


ETL e ELT

ETL (extração, transformação e carregamento) e ELT (extração, carregamento e transformação) são processos de processamento de dados usados para integrar informações em bancos de dados
Ferramentas:
Pentaho, DBT, e Airflow

  • DBT
  • Airflow
  • Pentaho
